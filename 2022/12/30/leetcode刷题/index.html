Python 中的 intersection（） 函数

Python set intersection（） 方法返回一个新集合，其中包含所有集合通用的元素

两个给定集合的交集是最大的集合，它包含两个集合共有的所有元素。两个给定集合 A 和 B 的交集是由 A 和 B 共有的所有元素组成的集合。

语法： set1.intersection（set2， set3， set4….）
参数：

  • 可以传递任意数量的集合

返回：返回一个集合，该集合具有所有集合（set1， set2， set3…）与 set1 的交集。仅当未传递任何参数时，它才返回 set1 的副本。

Python 中的 Union（） 函数

Python set Union（） 方法返回一个新集合，其中包含原始集合中的所有项。

两个给定集合的并集是包含两个集合的所有元素的集合。两个给定集合 A 和 B 的并集是一个集合，它由 A 的所有元素和 B 的所有元素组成，使得没有元素重复。

语法： set1.union（set2， set3， set4….）

参数：零套或多套

返回： 返回一个集合，该集合具有所有集合（set1， set2， set3…）与 set1 的并集。仅当未传递任何参数时，它才返回 set1 的副本。

Python 中的 sorted（） 函数

sorted（） 方法按升序或降序对给定序列以及 set 和字典（不是序列）进行排序（按字符对字符串字符进行 Unicode 比较），并始终返回排序列表。此方法不会影响原始序列。

语法：sorted(iteraable, key, reverse=False)affect*
参数：可迭代：
序列（列表，元组，字符串）或集合（字典，集合，冻结集）或任何其他需要排序的迭代器。
键（可选）：用作键或排序比较基础的函数。
反向（可选）：如果设置为 True，则可迭代对象将按反向（降序）顺序排序，默认情况下设置为 False。
返回类型：返回排序列表。

Python 中的 sort（） 函数

sort（） 函数与 sorted（） 非常相似，但与 sort 不同的是，它不返回任何内容并对原始序列进行更改。此外，sort（） 是一种列表类的方法，只能与列表一起使用。

语法：List_name.sort（key， reverse=False）
Parameters：*key：
一个用作排序比较键的函数。
反向：如果为 true，则列表按降序排序。
返回类型：无

sorted（）和sort（）默认都为升序排序。
